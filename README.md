## Запуск

Скопируйте файл с переменными окружения и, при необходимости, заполните значения:

```bash
cp .env.example .env
```

Запустите все сервисы:

```bash
docker compose up --build
```

После запуска доступно:
- Сервис: `http://localhost:8080`
- Симулятор провайдера: `http://localhost:8081`

## Запуск тестов

```bash
dotnet test
```

Тесты используют Testcontainers — Docker должен быть запущен.
